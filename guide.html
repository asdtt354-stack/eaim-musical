뮤지컬 메이커
Musical Maker
주제와 등장인물만 입력하면 Gemini AI가 대본·가사·Suno 프롬프트·배경 프롬프트를 자동으로 생성합니다.
01
Gemini API 키 입력 필수
선생님으로 입장 → 설정 탭에서 Gemini API 키를 입력하세요. 학생들은 API 키 없이 바로 사용할 수 있어요.
🔑 키 발급: aistudio.google.com → Get API key → Create API key (무료)
02
주제와 기본 정보 입력
주제는 짧게도 길게도 괜찮아요. 배경 장르(학교/가정/자연 등)와 분위기 장르(판타지/우정/코미디 등)를 순서대로 선택하고, 등장인물 수(2~20명)와 막 수(2~5막)를 설정합니다.
주제 예시:
· 짧게: "우정" "환경" "성장"
· 길게: "전학 온 아이가 음악을 통해 친구들과 하나가 되는 이야기"
03
✨ 뮤지컬 전체 생성 클릭
AI가 5단계로 자동 생성합니다. 제목 → 오프닝 코러스 → 각 막 → 커튼콜 → Suno 프롬프트 순서로 30초~1분 정도 소요됩니다.
04
결과 편집 및 활용
가사는 직접 클릭하여 수정할 수 있습니다. Suno 프롬프트를 복사하여 suno.ai에서 음악을 만들고, 배경 프롬프트로 이미지를 생성합니다.
다운로드: 대본(.txt), Suno 프롬프트(.txt) 별도 저장 가능
재생성: 마음에 안 들면 🔄 재생성 버튼으로 다시 만들 수 있어요
05
🎬 슬라이드쇼 열기 연동
하단의 "슬라이드쇼 열기" 버튼을 누르면 모든 장면 데이터(오프닝·각막·커튼콜 배경 프롬프트 포함)가 슬라이드쇼와 오디오 믹서에 자동으로 전달됩니다.
💡 수업 활용 팁
· 학생들이 직접 주제·등장인물을 정해서 생성해보게 하세요
· 생성된 가사를 수정하며 창작 수업으로 연결할 수 있어요
· 막 수는 수업 시간에 맞게 2막(짧은 수업)~5막(심화 수업)으로 조절하세요
· 크레딧에 학생 이름을 넣으면 더욱 의미있는 결과물이 됩니다
🎬
슬라이드쇼
Slideshow Player
AI 배경 이미지 + 2.5D 패럴랙스 + 캐릭터 애니메이션으로 공연 영상을 만들고 화면 녹화까지 할 수 있는 공연 스튜디오입니다.
01
API 키 설정 (첫 실행 시)
처음 열면 API 키 모달이 나타납니다. Gemini 키를 입력하면 AI가 각 장면의 배경 이미지를 생성하고, 건너뛰면 Pollinations AI로 대체됩니다. 뮤지컬 메이커에서 이미 입력했다면 자동으로 적용됩니다.
02
장면 탐색 — 키보드 / 버튼
하단 컨트롤 바 또는 키보드로 장면을 이동합니다.
→ 다음 장면
← 이전 장면
Space 다음 장면
ESC 메이커로 돌아가기
↑ W 가사 이전 줄
↓ S 가사 다음 줄
03
🎭 캐릭터 레이어 추가 NEW
하단 🎭 버튼 → 캐릭터 이미지 URL 입력 → 위치(왼쪽/오른쪽/가운데) 선택 → 적용. 마우스를 움직이면 배경과 캐릭터가 다른 속도로 움직여서 2.5D 입체감이 생깁니다.
패럴랙스 효과: 배경은 느리게, 캐릭터는 빠르게 움직여서 입체감 구현
마우스 없을 때: 자동으로 살살 부유하는 효과 적용
04
🎬 캐릭터 애니메이션 NEW
캐릭터 추가 후 🎭 버튼 → 🎬 애니메이션 버튼을 누르면 애니메이션 패널이 열립니다.
← 왼쪽 등장
→ 오른쪽 등장
↑ 아래서 등장
✦ 페이드인
🌊 둥둥 (상시)
💨 숨쉬기 (상시)
🍃 좌우흔들림 (상시)
⬆ 점프 (말할 때)
〰 흔들림 (말할 때)
→ 오른쪽 퇴장
✦ 페이드아웃
상시 효과: 버튼을 다시 누르면 토글로 ON/OFF 전환 가능
05
💬 말풍선 & 자막 기능
하단 💬 버튼을 누르면 말풍선 패널이 열립니다. 배우 이름과 대사를 입력하면 화면에 드래그 이동 가능한 말풍선이 생성됩니다.
말풍선: 꼬리 방향 선택 (↙ ↘ ↑), 드래그로 위치 이동, ✕ 버튼으로 삭제
자막: 📝 버튼 → 자막 텍스트 입력 → 화면 하단에 고정 표시
Enter키: 말풍선 빠르게 추가
06
⏺ 화면 녹화 → 영상 출력 NEW
하단 ⏺ 버튼을 누르면 화면 공유 창이 뜹니다. 화면을 선택하면 녹화가 시작되고, 좌측 상단에 빨간 녹화 인디케이터와 경과 시간이 표시됩니다. 다시 ⏹ 버튼을 누르면 .webm 영상 파일로 자동 저장됩니다.
활용법: 캐릭터 애니메이션 + 패럴랙스 + 말풍선이 모두 담긴 영상 제작 가능
저장 형식: .webm → CapCut에서 불러와 음악·효과음 추가 후 최종 완성
화면 공유: "전체 화면" 선택 권장
07
🌟 커튼콜 & 크레딧 롤
마지막 ▶ 버튼을 누르면 커튼콜 가사가 페이드인되고, 자동으로 크레딧 롤 (막 내림 자막)이 올라갑니다.
🎬 공연 화면 설정 팁
· 빔프로젝터/TV 연결 후 F11로 전체화면 → ESC로 나가기
· 배경 이미지 생성이 느리면 미리 한 번 전체 슬라이드를 돌려두세요 (캐시됨)
· 캐릭터 이미지는 배경 제거(누끼) 된 PNG 파일을 사용하면 더 자연스러워요
· 화면 녹화 후 CapCut에서 MR/AR 음악을 얹어 최종 영상 완성
🎚
오디오 믹서
Audio Mixer
BGM·MR·앰비언스 3채널 실시간 믹싱과 AI 자동 음향 프리셋으로 공연 현장의 음향을 완성합니다.
01
공연 전 — 큐리스트 준비 중요
뮤지컬 메이커에서 "슬라이드쇼 열기"를 누른 후 오디오 믹서를 열면 장면별 큐리스트가 자동 생성되어 있습니다.
큐리스트 확인: 오프닝→1막→2막→커튼콜 순서로 자동 생성
각 큐마다 AI가 장면 감정을 분석해 음향 프리셋(EQ·리버브·딜레이)을 미리 지정해 놓습니다
02
각 큐에 오디오 파일 연결
큐리스트에서 큐를 선택 → BGM 채널에 배경음악 업로드 → MR 채널에 반주 업로드 → "💾 현재 큐 저장" 클릭. 모든 막 반복 후 JSON 내보내기로 백업.
파일 형식: MP3, WAV 지원. 또는 URL 직접 입력도 가능
채널 구성: CH1 BGM(배경음악) / CH2 MR(반주) / CH3 앰비언스·나레이션
03
✦ AI 음향 프리셋 이해하기
상단 팔레트에서 9가지 분위기 프리셋 중 하나를 클릭하면 EQ·리버브·딜레이가 자동으로 최적화됩니다.
☀️ 밝고 신남 — 고음 부스트
🌧 슬프고 감동 — 저음↑, 리버브 강
⚡ 긴장·위기 — 저음↑↑, 빠른 딜레이
🌙 신비·환상 — 딜레이 깊음
🎭 공연장·극장 — 홀 리버브
🌟 커튼콜·피날레 — 웅장·여운
04
채널 컨트롤 사용법
각 채널에는 볼륨(VOL)·패닝(PAN)·페이드 시간(FADE)·수직 페이더가 있습니다.
▶ 재생
⏸ 일시정지
⏹ 정지·되감기
🔁 루프 ON/OFF
FADE↓ 서서히 줄이기
Space BGM 재생/정지
1~8 효과음 패드
05
공연 당일 — 큐 실행
큐리스트에서 장면 번호를 탭하면 파일 자동 교체 + AI 음향 프리셋이 동시에 적용됩니다. 재생 버튼만 누르면 됩니다.
🎚 음향 세팅 팁
· BGM은 루프(🔁 ON) 설정으로 장면이 끝날 때까지 반복 재생
· 장면 전환 전 FADE↓로 이전 음악을 서서히 줄이고 다음 큐 로드
· 마스터 볼륨으로 전체 출력을 85% 정도로 유지하면 안전합니다
· 공연 전 미리 전체 큐를 한 번 순서대로 테스트해 보세요
🎨
미디어픽쳐
Media Picture · 음악감상 창작 도구
음악을 듣고 느낀 감정을 한글로 표현하면 AI가 그림으로 만들어줍니다. 웹툰·수채화·지브리 등 다양한 스타일 선택 가능! 뮤직미디어레터 수업 모델의 핵심 도구입니다.
01
음악 감상 🎵
유튜브 URL을 붙여넣어 영상을 임베드하거나, 음악 파일(MP3/WAV)을 직접 업로드해서 재생합니다.
유튜브 탭: URL 입력 → ▶ 재생 → iframe으로 영상 임베드
파일 탭: MP3, WAV 업로드 → 파일명 표시 후 재생
Suno로 만든 음악 파일을 바로 올릴 수 있어요!
02
🎨 그림 스타일 선택
음악의 느낌에 맞는 그림 스타일을 선택합니다. 선택한 스타일이 자동으로 이미지 생성 프롬프트에 반영됩니다.
📱 웹툰 — 한국 만화체
🌸 지브리 — 부드러운 애니메이션
🎨 인상주의 — 고흐 스타일
💧 수채화 — 따뜻한 느낌
⭐ 애니메이션 — 일본 스타일
📸 실사영화 — 시네마틱
🔮 판타지 — 동화풍
👾 픽셀아트 — 레트로
🖋 수묵화 — 동양화풍
03
🌈 색감 & 💭 분위기 선택
음악에서 느껴지는 색깔과 감정을 여러 개 선택할 수 있습니다. 선택할수록 이미지가 더 구체적으로 생성됩니다.
색감: 빨강/주황/노랑/초록/파랑/보라/분홍 등 12가지
분위기: 신나는/슬픈/평온한/웅장한/설레는/어두운/그리운 등 10가지
04
✏️ 한글로 자유롭게 표현
음악을 듣고 떠오르는 장면을 한글로 자유롭게 적습니다. 🌐 영어로 번역 버튼을 누르면 AI가 자동으로 이미지 생성용 영어 프롬프트로 변환해줍니다.
예시: "가을 숲속에서 낙엽이 지고 있어. 혼자 걷는 느낌이야. 따뜻하면서도 조금 쓸쓸해..."
번역된 영어 프롬프트도 화면에 표시되어 학생들이 자연스럽게 영어를 배울 수 있어요!
05
✨ 이미지 생성 & 다운로드
✨ 이미지 생성하기 버튼을 누르면 Gemini AI가 선택한 스타일·색감·분위기·표현을 조합해 이미지를 생성합니다. 완성된 이미지는 바로 다운로드할 수 있습니다.
다시 생성: 🔄 버튼으로 같은 프롬프트로 다른 이미지 생성 가능
장면 추가: ➕ 버튼으로 여러 장면 이미지를 만들 수 있어요
06
🎬 CapCut으로 영상 완성
생성된 이미지를 다운로드하고 음악 파일도 저장한 후, CapCut에서 합쳐서 영상을 완성합니다.
1️⃣ 이미지 다운로드
2️⃣ 음악 파일 저장
3️⃣ CapCut 새 프로젝트
4️⃣ 이미지 + 음악 불러오기
5️⃣ 효과음·전환효과 추가
6️⃣ 완성! 🎉
💡 뮤직미디어레터 수업 활용 팁
· 같은 음악을 듣고 학생마다 다른 이미지가 나오는 것을 비교해보세요
· 스타일 선택 없이 자유 표현만으로도 충분히 좋은 이미지가 나와요
· 번역된 영어 프롬프트를 보여주며 영어 표현도 함께 배울 수 있어요
· 완성된 이미지 + 음악으로 학생 개인 뮤직비디오 제작까지 연결 가능!
🚦
신호등 연동 시스템
Slide Signal System
슬라이드쇼와 오디오 믹서가 신호로 소통합니다. 자동 실행이 아닌 음향 담당자가 직접 판단하고 실행합니다.
01
슬라이드쇼에서 신호 발신
슬라이드쇼에서 장면이 넘어갈 때마다 상단에 초록색 인디케이터가 표시됩니다. 장면 번호·제목이 오디오 믹서로 자동 전달됩니다.
02
오디오 믹서에서 신호 수신
오디오 믹서 우측 상단의 "🚦 슬라이드 연동 OFF" 버튼을 눌러 ON으로 켜두면 0.5초마다 신호를 감지합니다.
신호가 오면 오른쪽에 알림 카드가 표시됩니다:
· 장면 제목 — 어떤 장면인지
· AI 프리셋 — 분위기 이모지
· ▶ CUE XX 실행 — 클릭하면 해당 큐 즉시 로드
03
공연 역할 분담
두 사람이 각자 화면을 담당하면 진짜 공연처럼 운영할 수 있습니다.
🎬 슬라이드 담당 — 장면 전환, 가사 줄 넘기기
🎚 음향 담당 — 신호 확인 후 큐 실행, 볼륨 조절
같은 컴퓨터에서 두 탭으로 운영하거나, 노트북 두 대를 나란히 사용하세요.
같은 브라우저에서 두 탭을 열면 localStorage로 신호가 공유됩니다.
🌟
공연 당일 체크리스트
공연 전 이 순서대로 준비하면 실수 없이 진행할 수 있습니다.
D-1
하루 전 준비
✓ 뮤지컬 메이커에서 최종 버전 생성 완료
✓ 슬라이드쇼 전체 한 번 돌려서 배경 이미지 캐시 완료
✓ 캐릭터 이미지(누끼 PNG) 준비 및 URL 확인
✓ 오디오 믹서 — 모든 큐에 파일 연결 + JSON 저장 백업
✓ 효과음 패드 테스트
✓ 전체 리허설 (슬라이드 + 음향 동시 진행)
D-Day
공연 당일 순서
1. 슬라이드쇼 열기 → API 키 확인 → 커버 화면 대기
2. 오디오 믹서 열기 → 🚦 연동 ON → 큐 1(오프닝) 미리 로드
3. 마스터 볼륨 확인 (85% 권장)
4. 공연 시작 — 슬라이드 담당이 ▶ 클릭
5. 음향 담당 — 신호등 알림 보고 큐 실행
6. 커튼콜 → 오디오 믹서에서 커튼콜 큐 로드 → 웅장하게!
⚠️
비상 대처
배경 이미지 안 나올 때: 인터넷 확인 → 슬라이드 새로고침 → 건너뛰기 선택
음악 안 들릴 때: 오디오 믹서에서 채널 직접 클릭 → 마스터 볼륨 확인
신호등 안 올 때: 같은 브라우저인지 확인 → 연동 OFF 후 ON 다시 클릭
슬라이드 멈출 때: 키보드 → ← 로 수동 이동

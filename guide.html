뮤지컬 메이커
Musical Maker
주제와 등장인물만 입력하면 Gemini AI가 대본·가사·Suno 프롬프트·배경 프롬프트를 자동으로 생성합니다.
01
Gemini API 키 입력 필수
우측 상단 API 키 입력창에 Gemini API 키를 붙여넣고 저장 체크박스를 켜두세요. 다음에 자동 입력됩니다.
🔑 키 발급: aistudio.google.com → Get API key → Create API key (무료)
02
주제와 기본 정보 입력
주제는 짧게도 길게도 괜찮아요. 장르(학교생활, 판타지, 우정·성장 등), 막 수(2~5막), 등장인물을 설정합니다.
주제 예시:
· 짧게: "우정" "환경" "성장"
· 길게: "전학 온 아이가 음악을 통해 친구들과 하나가 되는 이야기"
등장인물: 민준(주인공), 서아(반장), 음악선생님 — 역할을 괄호로 표시하면 더 풍부하게 생성됩니다.
03
✨ 뮤지컬 전체 생성 클릭
AI가 5단계로 자동 생성합니다. 제목 → 오프닝 코러스 → 각 막 → 커튼콜 → Suno 프롬프트 순서로 30초~1분 정도 소요됩니다.
04
결과 편집 및 활용
가사는 직접 클릭하여 수정할 수 있습니다. Suno 프롬프트를 복사하여 suno.ai에서 음악을 만들고, 배경 프롬프트로 이미지를 생성합니다.
다운로드: 대본(.txt), Suno 프롬프트(.txt) 별도 저장 가능
재생성: 마음에 안 들면 🔄 재생성 버튼으로 다시 만들 수 있어요
05
🎬 슬라이드쇼 열기 연동
하단의 "슬라이드쇼 열기" 버튼을 누르면 모든 장면 데이터가 슬라이드쇼와 오디오 믹서에 자동으로 전달됩니다.
💡 수업 활용 팁
· 학생들이 직접 주제·등장인물을 정해서 생성해보게 하세요
· 생성된 가사를 수정하며 창작 수업으로 연결할 수 있어요
· 막 수는 수업 시간에 맞게 2막(짧은 수업)~5막(심화 수업)으로 조절하세요
· 크레딧에 학생 이름을 넣으면 더욱 의미있는 결과물이 됩니다
🎬
슬라이드쇼
Slideshow Player
AI 배경 이미지 위에 가사가 표시되고, 커튼콜·크레딧 롤까지 자동 재생되는 공연 영상 플레이어입니다.
01
API 키 설정 (첫 실행 시)
처음 열면 API 키 모달이 나타납니다. Gemini 키를 입력하면 AI가 각 장면의 배경 이미지를 생성하고, 건너뛰면 Pollinations AI로 대체됩니다.
한 번 저장해두면 다음부터는 자동으로 적용됩니다. 우측 상단 뱃지를 클릭하면 키를 변경할 수 있어요.
02
장면 탐색 — 키보드 / 버튼
하단 컨트롤 바 또는 키보드로 장면을 이동합니다. 점(dot) 네비게이터를 클릭해도 바로 이동할 수 있어요.
→ 다음 장면
← 이전 장면
Space 다음 장면
ESC 메이커로 돌아가기
↑ W 가사 이전 줄
↓ S 가사 다음 줄
03
가사 자막 제어
화면 하단에 현재 줄·다음 줄 가사가 노란색으로 표시됩니다. ↑↓ 키로 가사 줄을 수동으로 넘길 수 있어요.
가사 숨기기: 우측 하단 "가사 숨기기" 버튼으로 자막을 끄고 켤 수 있습니다
04
💬 말풍선 & 자막 기능
하단 컨트롤의 💬 버튼을 누르면 말풍선 패널이 열립니다. 배우 이름과 대사를 입력하면 화면에 드래그 이동 가능한 말풍선이 생성됩니다.
말풍선: 꼬리 방향 선택 (↙ ↘ ↑), 드래그로 위치 이동, ✕ 버튼으로 삭제
자막: 📝 버튼 → 자막 텍스트 입력 → 화면 하단에 고정 표시
Enter키: 말풍선 빠르게 추가
05
⏵ 자동재생 모드
⏵ 버튼을 누르면 7초마다 자동으로 다음 장면으로 넘어갑니다. 다시 누르면 멈춥니다.
공연 연습 시에는 자동재생, 실제 공연 시에는 수동 제어를 추천합니다
06
🌟 커튼콜 & 크레딧 롤
마지막 ▶ 버튼을 누르면 커튼콜 가사가 페이드인되고, 자동으로 크레딧 롤 (막 내림 자막)이 올라갑니다.
🎬 공연 화면 설정 팁
· 빔프로젝터/TV 연결 후 F11로 전체화면 → ESC로 나가기
· 배경 이미지 생성이 느리면 미리 한 번 전체 슬라이드를 돌려두세요 (캐시됨)
· 인터넷이 불안정한 환경에서는 Pollinations(건너뛰기)를 추천합니다
· 장면 전환 시 자동으로 암전 효과가 적용됩니다
🎚
오디오 믹서
Audio Mixer
BGM·MR·앰비언스 3채널 실시간 믹싱과 AI 자동 음향 프리셋으로 공연 현장의 음향을 완성합니다.
01
공연 전 — 큐리스트 준비 중요
뮤지컬 메이커에서 "슬라이드쇼 열기"를 누른 후 오디오 믹서를 열면 장면별 큐리스트가 자동 생성되어 있습니다.
큐리스트 확인: 오프닝→1막→2막→커튼콜 순서로 자동 생성
각 큐마다 AI가 장면 감정을 분석해 음향 프리셋(EQ·리버브·딜레이)을 미리 지정해 놓습니다
02
각 큐에 오디오 파일 연결 (방식 A)
큐리스트에서 큐를 선택 → BGM 채널에 그 막의 배경음악 업로드 → MR 채널에 반주 업로드 → "💾 현재 큐 저장" 클릭. 모든 막 반복 후 JSON 내보내기로 백업.
파일 형식: MP3, WAV 지원. 또는 URL 직접 입력도 가능
큐 로드 시: 파일이 자동으로 교체됨 — 막마다 다른 음악 자동 전환
채널 구성: CH1 BGM(배경음악) / CH2 MR(반주) / CH3 앰비언스·나레이션
03
✦ AI 음향 프리셋 이해하기
상단 팔레트에서 9가지 분위기 프리셋 중 하나를 클릭하면 EQ·리버브·딜레이가 자동으로 최적화됩니다. 볼륨은 직접 제어합니다.
☀️ 밝고 신남 — 고음 부스트, 잔향 없음
🌧 슬프고 감동 — 저음↑, 리버브 강
⚡ 긴장·위기 — 저음↑↑, 빠른 딜레이
🌙 신비·환상 — 고음↑, 딜레이 깊음
🎭 공연장·극장 — 홀 리버브
🌟 커튼콜·피날레 — 웅장·여운
04
채널 컨트롤 사용법
각 채널에는 볼륨(VOL)·패닝(PAN)·페이드 시간(FADE)·수직 페이더가 있습니다. EQ 커브가 실시간으로 시각화됩니다.
▶ 재생
⏸ 일시정지
⏹ 정지·되감기
🔁 루프 ON/OFF
FADE↓ 서서히 줄이기
FADE↑ 서서히 키우기
Space BGM 재생/정지
1~8 효과음 패드
05
⚡ 효과음 패드
우측 패드에 박수·천둥·바람·종소리 등 기본 효과음이 사전 탑재되어 있습니다. 패드를 클릭하거나 키보드 1~8 키로 즉시 재생합니다.
패드 위에 파일을 업로드하면 원하는 효과음으로 교체 가능
+ 효과음 추가 버튼으로 패드를 늘릴 수 있습니다
06
공연 당일 — 큐 실행
큐리스트에서 장면 번호를 탭하면 파일 자동 교체 + AI 음향 프리셋이 동시에 적용됩니다. 재생 버튼만 누르면 됩니다.
각 큐에 연결된 파일명이 🎵/🎼 배지로 표시됩니다
자동 ON: 🤖 자동 적용 ON 상태에서 큐 로드 시 AI 프리셋 자동 전환
🎚 음향 세팅 팁
· BGM은 루프(🔁 ON) 설정으로 장면이 끝날 때까지 반복 재생
· 장면 전환 전 FADE↓로 이전 음악을 서서히 줄이고 다음 큐 로드
· 마스터 볼륨으로 전체 출력을 85% 정도로 유지하면 안전합니다
· 공연 전 미리 전체 큐를 한 번 순서대로 테스트해 보세요
🚦
신호등 연동 시스템
Slide Signal System
슬라이드쇼와 오디오 믹서가 신호로 소통합니다. 자동 실행이 아닌 음향 담당자가 직접 판단하고 실행합니다.
01
슬라이드쇼에서 신호 발신
슬라이드쇼에서 장면이 넘어갈 때마다 상단에 초록색 인디케이터가 표시됩니다. 장면 번호·제목이 오디오 믹서로 자동 전달됩니다.
02
오디오 믹서에서 신호 수신
오디오 믹서 우측 상단의 "🚦 슬라이드 연동 OFF" 버튼을 눌러 ON으로 켜두면 0.5초마다 신호를 감지합니다.
신호가 오면 오른쪽에 알림 카드가 표시됩니다:
· 장면 제목 — 어떤 장면인지
· AI 프리셋 — 분위기 이모지
· ▶ CUE XX 실행 — 클릭하면 해당 큐 즉시 로드
03
공연 역할 분담
두 사람이 각자 화면을 담당하면 진짜 공연처럼 운영할 수 있습니다.
🎬 슬라이드 담당 — 장면 전환, 가사 줄 넘기기
🎚 음향 담당 — 신호 확인 후 큐 실행, 볼륨 조절
같은 컴퓨터에서 두 탭으로 운영하거나, 노트북 두 대를 나란히 사용하세요.
같은 브라우저에서 두 탭을 열면 localStorage로 신호가 공유됩니다.
🌟
공연 당일 체크리스트
공연 전 이 순서대로 준비하면 실수 없이 진행할 수 있습니다.
D-1
하루 전 준비
✓ 뮤지컬 메이커에서 최종 버전 생성 완료
✓ 슬라이드쇼 전체 한 번 돌려서 배경 이미지 캐시 완료
✓ 오디오 믹서 — 모든 큐에 파일 연결 + JSON 저장 백업
✓ 효과음 패드 테스트
✓ 전체 리허설 (슬라이드 + 음향 동시 진행)
D-Day
공연 당일 순서
1. 슬라이드쇼 열기 → API 키 확인 → 커버 화면 대기
2. 오디오 믹서 열기 → 🚦 연동 ON → 큐 1(오프닝) 미리 로드
3. 마스터 볼륨 확인 (85% 권장)
4. 공연 시작 — 슬라이드 담당이 ▶ 클릭
5. 음향 담당 — 신호등 알림 보고 큐 실행
6. 커튼콜 → 오디오 믹서에서 커튼콜 큐 로드 → 웅장하게!
⚠️
비상 대처
배경 이미지 안 나올 때: 인터넷 확인 → 슬라이드 새로고침 → 건너뛰기 선택
음악 안 들릴 때: 오디오 믹서에서 채널 직접 클릭 → 마스터 볼륨 확인
신호등 안 올 때: 같은 브라우저인지 확인 → 연동 OFF 후 ON 다시 클릭
슬라이드 멈출 때: 키보드 → ← 로 수동 이동
